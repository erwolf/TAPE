# README

HOW TO:

You need to clone the repo into your RailsProject folder. The repository is the entire Rails app, so you don't have to worry about rails new or anything.

Each time you pull changes you need to run 'bundle install'


GIT DEV CYCLE
- pull changes from the repo:
git pull https://github.com/erwolf/TAPE

- make changes and add the files to push
git add <filenames>

- commit changes
git commit -m "Helpful commit message"

- push upstream and pass your credentials
git push
